Detailed Description

This file holds a very basic code for LPC1769. This code configures Flash wait-states and enables the PLL0 to achieve the highest allowed frequency for LPC1769 (120MHz). Main code block just blinks the LED. The LED port and pin are defined in config.h file. Target core frequency and quartz crystal resonator frequency are defined there as well.
